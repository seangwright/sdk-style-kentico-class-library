# SDK Style Kentico Class Library

Examples of a Kentico class library and unit test project using the new .NET Core SDK style project system

## Requirements

Install the following development requirements if you have not already

- [.NET Framework 4.7.2 Developer Pack](https://dotnet.microsoft.com/download/thank-you/net472-developer-pack)
- [.NET Core SDK >=2.2.100 <2.2.203](https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-2.2.106-windows-x64-installer)

## Projects

| Name                             | Description                                             | TFM             |
| -------------------------------- | ------------------------------------------------------- | --------------- |
| Kentico.EcommerceUtilities       | Simple ecommerce related utility classes for Kentico 12 | netframework472 |
| Kentico.EcommerceUtilities.Tests | Unit test project for Kentico.EcommerceUtilities        | netframework472 |
